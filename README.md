# vacancy-analytics

Програмна система для збору, обробки та OLAP-аналізу вакансій. Проєкт виконує парсинг даних про вакансії, очищення та підготовку даних, аналіз основних характеристик вакансій і формування графічних результатів.

## Функціональність

* збір даних про вакансії;
* збереження сирих даних у CSV-файл;
* очищення та попередня обробка даних;
* аналіз вакансій за містами;
* аналіз зарплат;
* аналіз вимог до освіти та досвіду;
* аналіз віддаленого формату роботи;
* аналіз навичок, які найчастіше зустрічаються у вакансіях;
* побудова графіків і збереження результатів у файли.

## Встановлення

Клонувати репозиторій:

```bash
git clone <посилання-на-репозиторій>
```

Перейти в папку проєкту:

```bash
cd <назва-папки-проєкту>
```

Створити та активувати віртуальне середовище:

```bash
python -m venv .venv
.venv\Scripts\activate
```

Встановити залежності:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Вхідні дані

Вхідними даними є інформація про вакансії, яка збирається програмою та зберігається у папці `data`.

## Вихідні дані

Після запуску програма формує результати у папці `output`.

До результатів належать CSV-файли з обробленими даними, підсумкові таблиці OLAP-аналізу, а також графіки розподілу вакансій за містами, зарплатами, досвідом, освітою, форматом роботи та навичками.

## Результат роботи

У результаті виконання програми формується набір матеріалів для аналізу ринку вакансій: очищений набір даних, аналітичні таблиці, графіки та короткий підсумковий звіт.
